#### Arquitetura limpa .NET Core 8 com xUnit

Uma API em .NET Core 8.0.x com foco em testes com o xUnit.

Foi aplicado os principios do SOLID e código limpo.

 O que foi abordado nesse projeto:
 > SOLID
> 
 > Clean Code
> 
 > xUnit
> 
 > Testes Unitarios
> 
 > Testes de integração
> 
 > Moq e AutoMock
> 
> Fixture
> 
 > SQLite
